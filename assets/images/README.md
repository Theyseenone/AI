# Login Icon

Please upload your login icon image here and name it `login_icon.png`.

## Supported Formats
- PNG (recommended)
- JPG/JPEG
- GIF
- WebP
- BMP
- WBMP

## Recommended Specifications
- Size: At least 80x80 pixels for crisp display
- Format: PNG with transparent background if needed
- File name: `login_icon.png`

Once you upload the image, the login form will automatically display it instead of the default chat bubble icon.
